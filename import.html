Import / Update Parts Master

Smart upsert — if a part number already exists in the system it will be updated with the new data. If it's a new part number it will be created. This way you can fix rates, rename parts, or add new ones all in one go.

1
Download current partsDownloads all existing parts pre-filled so you can edit them directly.
2
Edit in Excel / Google SheetsUpdate rates, fix names, add new rows. Part number is the key — don't change it if you want to update an existing part.
3
Upload & importPreview shows each row as NEW or UPDATE before anything is saved.
Column guide:
part_no — Part number, required. This is the key — existing part numbers get updated  |  name — Part name, required  |  customer — Customer code (e.g. PVPL, ATUL, M&M)  |  rate — Rate in ₹, can be 0

3
